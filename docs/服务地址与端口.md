# 服务地址与端口一览

> 本文档汇总本工程对外常用的 **IP/主机名、端口、URL** 及用途。  
> **权威配置**以 `ptz_config.yaml`、`mediamtx.yml` 为准；若修改端口，请同步更新本文。

---

## 1. 主机占位说明

| 占位符 | 含义 | 示例 |
|--------|------|------|
| `<本机>` | 仅本机访问 | `127.0.0.1` 或 `localhost` |
| `<局域网IP>` | 同网段其它设备访问本机 | 如 `192.168.1.100`（用 `ip a` / `hostname -I` 查看） |

下文 URL 中统一写为 **`<HOST>`**：本机调试时替换为 `127.0.0.1`，外网/局域网设备替换为 `<局域网IP>`。

---

## 2. 当前默认端口（与 `ptz_config.yaml` 一致）

| 端口 | 协议 | 进程/组件 | 说明 |
|------|------|-----------|------|
| **6670** | TCP | `ptz_launcher.py` | Web 控制台、ONVIF SOAP（HTTP）、REST/WS 入口 |
| **6671** | TCP | `ptz_stream.py`（仿真运行时） | Isaac 内嵌 HTTP；一般经 6670 代理，可不对外暴露 |
| **6674** | TCP | MediaMTX | RTSP 拉流 `…/ptz_cam` |
| **3702** | UDP | `ptz_launcher.py` | WS-Discovery 多播（`239.255.255.250:3702`） |
| **6672** | TCP | `tls_tcp_proxy.py`（**可选**，需自行启动） | HTTPS/WSS → 转发到 6670 |

## 3. 浏览器 / HTTP / WebSocket（端口 6670）

| 用途 | 地址 |
|------|------|
| Web 控制台 | `http://<HOST>:6670/` |
| 状态 JSON | `http://<HOST>:6670/status` |
| Isaac 日志片段 | `http://<HOST>:6670/log` |
| 单帧快照（ONVIF 同源） | `http://<HOST>:6670/onvif-snap.jpg` |
| WebSocket JPEG | `ws://<HOST>:6670/ws` |
| WebSocket FLV（H.264） | `ws://<HOST>:6670/ws-flv` |
| 场景状态（代理） | `http://<HOST>:6670/scene/state` |

**POST（控制，需 Content-Type 等按接口要求）**

| 路径 | 功能 |
|------|------|
| `http://<HOST>:6670/start` | 启动仿真 |
| `http://<HOST>:6670/stop` | 停止仿真 |
| `http://<HOST>:6670/control` | PTZ JSON |
| `http://<HOST>:6670/scene/gondola` | 吊篮场景 |
| `http://<HOST>:6670/scene/workers` | 工人数量 |

## 5. Isaac 直连（端口 6671，仅仿真运行时）

> 外部优先使用 **6670**；下表供本机调试或防火墙仅开 6671 时参考。

| 用途 | 地址 |
|------|------|
| 状态 | `http://<HOST>:6671/status` |
| MJPEG | `http://<HOST>:6671/stream.mjpeg` |
| 快照 | `http://<HOST>:6671/snapshot.jpg` |
| 场景状态 | `http://<HOST>:6671/scene/state` |
| PTZ / 场景 POST | 同路径名，与 6670 代理一致（`/control`、`/scene/*`） |

## 6. RTSP（端口 6674）

| 用途 | 地址 |
|------|------|
| 主码流（H.264） | `rtsp://<HOST>:6674/ptz_cam` |

播放示例（本机、TCP）：参见仓库 `scripts/play_rtsp.sh` 或：

`ffplay -rtsp_transport tcp -fflags nobuffer -flags low_delay rtsp://127.0.0.1:6674/ptz_cam`

---

## 7. 可选 HTTPS/WSS（端口 6672）

若已按 `README_local.md` 启动 `tls_tcp_proxy.py`：

| 用途 | 地址 |
|------|------|
| 控制台 | `https://<HOST>:6672/` |

证书为自签名时浏览器需信任/放行。
